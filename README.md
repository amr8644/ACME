# Social Media

A CRUD app that allows user's to post, update delete posts.

## Demo

Live Preview

[Live](https://social-media-amr8644.vercel.app/)

## Tech Stack

**Client:** Next JS and Chakra UI

**Server:** Golang and PostgreSQL

## Run Locally

Clone the project

```bash
  git clone https://github.com/amr8644/Social-Media.git
```

Go to the project directory

```bash
  cd Social-Media
```

## For Client Side

```bash
  cd client
```

Install dependencies

```bash
  yarn install
```

Run

```bash
  yarn run dev
```

## For Server Side

```bash
  cd server/cmd
```

Build project

```bash
  go build main.go -o main
```

Start server

```bash
./main
```
