# Social Media

A CRUD app that allows user's to post, update delete posts.

## Tech Stack
**Server:** Golang and PostgreSQL

## Run Locally

Clone the project

```bash
  git clone https://github.com/amr8644/Social-Media.git
```

Go to the project directory

```bash
  cd Social-Media
```
## For Server Side

```bash
  cd server/cmd
```

Build project

```bash
  go build main.go -o main
```

Start server

```bash
./main
```
